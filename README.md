## Project Overview
This project gradually develops a complete understanding of the Multilayer Perceptron:
###  Task 1 – Forward Pass
Manual computation of weighted sums and activations.
### Task 2 – Loss Computation
Manual MSE loss formula on small dataset.
### Task 3 – Backpropagation
Manual gradient computation using the chain rule.
###  Task 4 – Weight & Bias Updates
Applying learning rate and update equations.
### Task 5 – Gradient Descent
Performing full update cycles manually.
###  Task 6 – Optimizer Methods
Discussing Momentum, NAG, RMSProp, Adam (theory only).

### MLP Implementation (NumPy Only)
A complete NumPy neural network with:
- 784 → 128 → 64 → 10 architecture  
- Sigmoid activations  
- Mini-batch Gradient Descent  
- MNIST dataset  
- Loss curve  
- Test accuracy  
- Sample predictions  

###  Report (LNCS Format)
A scientific write-up explaining:
- MLP theory  
- Forward pass  
- Backpropagation  
- Optimizer equations  
- Experimental results on MNIST
